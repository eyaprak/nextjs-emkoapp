# React.JS Movie App

## General info
This project has made with `Next.js` framework. This app powered by `jsonplaceholder` fake api to get user informations.

## Setup
To run this project, install it locally using npm:
```
$ cd ../nextjs-emkoapp
$ npm install
$ npm run dev
```
